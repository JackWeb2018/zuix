### Install ### Quick start The juice global object consists of a few methods: - juice.field - juice.include - juice.view - juice.controller - juice.load #### Declaring a View template Before applying a view template to a jUIce component, it is required to declare it by assigning an unique identifier. A view template can be either placed inline in the document or it can be loaded from a separate HTML snippet file placed on a remote server. ##### Inline View template Inline view template definition is achieved by adding a ```data-ui-view``` attribute to the main container element of the view template. The value of this field is meant to be an unique identifier. If the same value is used for more than one element, the last declaration will override any other previous one.

NOTE: jUIce will first look for an inline template matching the ```data-ui-view``` value. If none found it will try to load it from a remote resource by appending ```.html``` suffix to the template name. In the following example:
The template lookup order will be: 1. An inline view template with ```data-ui-view = ui/badges/cool_one``` or ```data-ui-view = http://remote.server/jcdn/ui/badges/cool_one``` (inline overriding of a remote template) 2. A remote template file located on the same server if a relative path was given as the value of the ```data-ui-load``` attribute, or any other remote server if an absolute url was given the first found template resource will be applied to the component.
