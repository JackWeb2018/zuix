Content dashboard Manager

Simplifies site structuring and management, with content inclusion feature, content and style files can be conveniently separated.
Optimizes loading speed with lazy-loading feature, loading content only when it actually comes into the user's screen view.
Can load and reuse content from across sites with just one HTML tag.

Template settings_ethernet Engine

Can turn any HTML snippet into a template. No custom tags or "mustaches" needed, so that the templates preview is immediate and real.
Automatic fields mapping and content rendering when applying a template to an element, or custom binding-adapters with full control over model-view mapping process.
