### jUIce API documentation The ```juice``` global object exposes the following methods: #### ```juice.load( name [, options] )``` Loads a component and returns the associated ```{ComponentContext}``` object. This method will try in order: - succesfully get the view associated to the component ```name``` or fire the ```options.error``` callback on failure - if previous step was succesfull it will cache the view (other components may refer the same view) - then it will try to load a {ContextController} either from the inline ```options.controller``` or from the ```name```.js 
Method summary
    {ComponentContext} juice.load( view [, options] )

Parameters
    {string|jQuery|HTMLElement} view
        Mixed type parameter that can be either one of the following:
        - a ```string``` with the value matching the ```data-ui-view``` attribute
          of an inline element in the document
        - a ```string``` indicating the relative or absolute path of the view file
          (without the .html extension)
        - a ```{jQuery}``` $(element)
        - a ```HTMLElement```
    {ContextOptions} options
        Options object used to initialize the component context.

Returns
    {ComponentContext}
    The loaded component context.
#### ```juice.controller( controller_impl )``` Initializes a controller function. 
Method summary
    {ContextController} juice.controller( controller_impl )

Parameters
    {ContextController} controller_impl
    The context controller function.

Returns
    {ContextController}
    The initialized controller.

$$ is the {ComponentContext} object that is passed to the {ContextController} function once the related component is created. When calling the juice.controller(...) method, the passed {ContextController} function can implement none, one or more of the $$ context handlers. ##### $$ {ComponentContext} object summary

#### ```juice.field( name [, container])``` Returns the ```{jQuery}``` object of the element having ```data-ui-field``` attribute matching ```name```. If no ```container``` specified the search range will cover the entire document. 
Method summary
    {jQuery} juice.field( name [, container] )

Parameters
    {string} name
        ...
    {string|jQuery|HTMLElement} container
        A valid jQuery element/selector.

Returns
    {jQuery}
    The {jQuery} element with ```data-ui-field``` matching ```name```.
