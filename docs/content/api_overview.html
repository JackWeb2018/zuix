### API The ```zuix``` global object exposes the following methods: #### ```zuix.controller( controller_impl )``` Initializes a controller function. 
Method summary

Parameters
    {ContextController} controller_impl
    The context controller function.

Returns
    {ContextController}
    The initialized controller.
#### ```zuix.field( name [, container])``` Returns the ```{ZxQuery}``` object of the element having ```data-ui-field``` attribute matching ```name```. If no ```container``` specified the search range will be the entire document. 
Method summary
    {ZxQuery} zuix.field( name [, container] )

Parameters
    {string} name
        ...
    {string|ZxQuery|HTMLElement} container
        A valid ZxQuery element/selector.

Returns
    {ZxQuery}
    The {ZxQuery} element with ```data-ui-field``` matching ```name```.

#### ```zuix.load( name [, options] )``` Loads a component and returns the associated ```{ComponentContext}``` object. 
Method summary
    {ComponentContext} zuix.load( view [, options] )

Parameters
    {string|ZxQuery|HTMLElement} view
    Mixed type parameter that can be either one of the following:
    - a ```string``` with the value matching the ```data-ui-view```
      attribute of an inline element in the document
    - a ```string``` indicating the relative or absolute path
      of the view file (without the .html extension)
    - a ```{ZxQuery}``` $(element)
    - a ```HTMLElement```
    {ContextOptions} options
        Options object used to initialize the component context.

Returns
    {ComponentContext}
    The loaded component context.
